Automation Application August 2024

Developed a web application replicating core functionalities of Zapier using Next.js for full-stack development and Tailwind CSS for responsive design. Implemented a webhook service with Node.js and Express to efficiently handle incoming requests, and integrated Kafka for asynchronous communication between a sweeper service that pushes triggers to a queue and a worker service that processes these triggers. Utilized PostgreSQL as the database, employing Prisma as the ORM to streamline database interactions.

Digital Wallet Application July 2024

Created a comprehensive digital wallet application enabling users to transfer funds between their bank accounts and digital wallet, as well as transfer from one user to another user (digital wallet). The project was implemented using Next.js for both the frontend (also used Tailwind CSS for styling) and backend for the user application, while a separate Node.js server handled bank webhooks to confirm successful bank transfers. Data management was facilitated through a PostgreSQL database with Prisma as the ORM, and user authentication was managed with Next Auth. The entire codebase was organized as a monorepo using Turborepo as a build system.

Asset Exchange Application June 2024

Developed an asset exchange application like Coinbase, featuring a dynamic order book that allows users to place buy (bids) and sell (asks) orders. The frontend was built using Next.js for a responsive user experience, while the backend was developed with Node.js to efficiently manage order processing and user interactions. Employed Redis for pub/sub messaging and enabled real-time communication through WebSocket integration to ensure seamless updates to the order book on the front-end.

Enhanced Form Builder Application April 2024

Developed an advanced form creation and analytics tool using the MERN stack, featuring React for the frontend, Node.js and Express for the backend, and MongoDB for the database. An additional feature allows categorizing users into different groups and sub-groups, enabling comparative analysis of responses across these different groups. The entire application was deployed using Vercel.
